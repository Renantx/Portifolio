Estágio no TCU (Tribunal de Contas da União) como desenvolvedor Oracle Application Express (APEX).

Data: 2020 - Atualmente.
Linguagem: SQL, CSS, HTML e Javascript.

Descrição:

Desenvolvimento de aplicações na plataforma Oracle APEX. Como a plataforma é baseada em um banco de dados, todas as aplicações devem ser pensadas e estruturadas através de tabelas, views, índices, sequências, tipos, pacotes, procedimentos, funções, triggers, etc...

Levando em consideração que o trabalho prestado lá é de natureza confidencial, não estarei descrevendo as aplicações desenvolvidas por motivo de sigilo.

Desenvolvimento de chatbot sobre redes de computadores

Data: 2021.
Linguagem: Prolog

Descrição:

Projeto desenvolvido na disciplina Projeto Integrador em Ciências da Computação IV, da Universidade Cruzeiro do Sul.

O projeto consistia em construir um chatbot especialista em responder perguntas sobre redes de computadores. Essa construção foi dividida em 4 etapas:

  • Construir base de dados 1
  • Construir base de dados 2
  • Criar regras para realização das consultas
  • Implementação do chatbot em prolog

A ferramenta utilizada para realização do projeto foi SWI-prolog e os arquivos do projeto (github), a apresentação (vídeo) demonstrando o funcionamento do chatbot e o site do SWI-prolog (ferramenta ultilizada), podem ser conferidos nos links abaixo.

Links: Github / Apresentação / SWI-Prolog

Representação gráfica de dados do Covid-19 no Brasil em 02/06/2020

Data: 2020.
Linguagem: Python

Descrição:

Projeto desenvolvido na disciplina Vizualização da Informação, da Universidade Cruzeiro do Sul.

O projeto consistia em fazer uma representação gráfica (em python) de um dataset de minha escolha. O dataset escolhido foi o de casos de Covid-19 no Brasil no dia 02/06/2020, retirado do site do Ministério da Sáude do Brasil.

O desenvolvimento do projeto foi feito através da ferramenta Jupyter Notebook e os arquivos do projeto (github), a apresentação (vídeo) explicando o porque das escolhas dos tipos de gráficos, o site do Ministério da Saúde (onde os dados foram retirados) e o site do Jupyter Notebook (ferramenta utilizada), podem ser acessados nos links abaixo.

Links: Github / Vídeo / Ministério da Saúde / Jupyter Notebook
